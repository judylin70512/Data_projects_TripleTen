# Superstore Returns Analysis (Tableau)

## Business Problem

Product returns can significantly impact retail profitability.  
This project analyzes the root causes of returned orders in the Superstore dataset and identifies patterns that may help reduce return rates.

## Analysis

Multiple perspectives were analyzed to understand return behavior:

- Return rate by product category
- Customer return patterns
- Geographic distribution of returns
- Seasonal return trends

## Dashboard

The Tableau dashboard allows users to explore return patterns interactively.
